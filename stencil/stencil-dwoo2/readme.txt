This plugin enables the use of the Dwoo 2 template engine in combination with the Stencil plugin.

== Description ==
The Stencil: Dwoo 2 Implementation enables the use of the Dwoo template engine in your themes.

For specific details about how your theme should be set up, please see the installation instructions of the Stencil plugin.
You can see a working sample theme at <https://github.com/moorscode/stencil-sample-theme-dwoo2>.

If you are familiar with a template engine other than Dwoo 2 check out the other Implementations:

* Dwoo 2
* Mustache
* Savant 3
* Smarty 2.x
* Smarty 3.x
* Twig

== Installation ==
1. Upload the folder `stencil-dwoo2` to the `/wp-content/plugins/` directory or use the WordPress build in plugin-installer
1. Make sure you have installed and activated the 'Stencil' plugin
1. Activate the plugin through the 'Plugins' menu in WordPress
